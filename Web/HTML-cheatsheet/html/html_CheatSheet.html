Dentro del head

Dentro del <head> se encuentra toda la metada data de la página: título. codificación de caracteres, etc.

¿Qué son los header?

Las etiquetas <h> siempre van acompañadas de un número que modifica el tamaño de la fuente-

h1

h2

h3

h4

h5

Sirven para indicar titulos de secciones y jerarquizar la información

Uso de los div

Los <div> sirven para crear secciones y agrupar información. Dentro se pueden usar los encabezados y los parrafos de texto: demarcados por la etiqueta <p>
La etiqueta <span> tiene la misma función que el <div> y puede ser usado de la misma forma.

Listas

Lista no ordenada

  • Para iniciarlizar el bloque de lista se usa la etiqueta <ul>
  • Se indican los valores de la lista usando <li>
  • Los valores son representados usando bullets points

Lista ordenada

  1. Para inicializar el bloque de lista ordenada se usa la etiqueta <ol>
  2. Al igual que en la lista no ordenad, los valores se indican usando <li>
  3. Los valores de la lista son representados usando números

Tablas

Se inicializa la tabla usando la etiqueta <dtable> después, se define los encabezados con <dthead> y finalmente se añaden los elementos de las celdas con <td> por filas con <tr>
