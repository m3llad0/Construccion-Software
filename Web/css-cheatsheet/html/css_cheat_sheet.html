Propiedad border

La propiedad border permite definir reglas para todos los bordes seleccionados. Con border se pueden definir los siguientes valores: border-width, border-style, border-color.

border-left, border-right, border-top, border-bottom

Como su nombre lo indica, sirve para modificar las propiedades antes mencionadas de los bordes izquierdo, derecho, superior o inferior.

Border radius

Sirve para redondear los borders de un bloque.

Background image

Como su nombre lo indica, sirve para colocar una imagen de fondo

Color

Sirve para cambiar el color de los elementos de texto de un bloque

Font

Sirve para cambiar las propiedades de un elementos de una fuente: tamaño, variante, etc.

Font family

Sirve para cambiar la 'familia' de una fuente para modificar de forma integral titulos y texto.

font-style, font-weight

Sirve para cambiar propiedades de la fuente: como el estilo y el 'peso' qué tan bold es la fuente.

width, height

Modifica las propiedades de anchura y altura de los bloques.

List style

Sirve cambiar el estilo de una lista

Margin

Sirve para fijar el area del margen de los cuatro lados.

Padding

Sirve para modificar el area de padding de los cuatro extremos de un bloque.

Overflow

Sirve para modificar la forma en como fluye el texto en un bloque.

Display

Especifica si un elemento es un blockinline or inline element y el diseño usado por sus hijos como diseño de flujo.

Position

Especifica cómo un elemento es poscionado en el documento.

Filter

Sirve para dar efectos graficos como desenfoque o cambio de color.
